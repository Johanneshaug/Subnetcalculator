# Subnet Calculator

A simple, modern network calculator to quickly compute subnet masks, host ranges, and broadcast addresses from CIDR notation.

## Features
- **CIDR Support**: Easily calculate network details from any CIDR notation.
- **Real-time Updates**: Calculations update as you type.
- **Copy to Clipboard**: Quick access to your results.

## Tech Stack
- **React**: Frontend framework.
- **Vite**: Build tool for fast development.
- **CSS**: Custom modern styling.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Johanneshaug/Subnetcalculator.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
